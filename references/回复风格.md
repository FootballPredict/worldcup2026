# 回复风格

答案要简单易读但格式美观。先给结论，简短支撑，然后停止。以推理为主，诚实对待不确定性。

## 原则
- 先给核心答案：倾向、比分或赔率。
- 行文简短。一两句推理胜过一段话。
- 概率给诚实的整数或区间（如 `~55%`、`20-25%`）。不要捏造没有计算过的小数。
- 比分预测默认给两个候选比分，并给出每个比分的大致概率；不要只写“备选”而不写概率。
- 多结果用小表格或短标签列表。不要深层嵌套，不要大段文字。
- 点名2-3个真实驱动因素（档位差、主场优势、伤停），让结论透明而非黑箱。
- 始终以概率框架表达结果，热门随时会输。

## 单场比赛回答模板
```
美国 vs 澳大利亚（小组赛）
胜平负：美国胜 ~52% | 平局 ~24% | 澳大利亚胜 ~24%
比分：1-0（~17%） | 1-1（~12%）
理由：美国高出约半档，叠加东道主主场；澳大利亚防守有组织，所以首选小胜，平局仍有现实概率。
依据：Elo/FIFA评分、近期状态、伤停与场地因素；具体名单以最新消息为准。
```

## 多场比分模板
```
北京时间6月24日预测
| 时间 | 比赛 | 胜平负倾向 | 两个比分 |
|---|---|---|---|
| 04:00 | 英格兰 vs 加纳 | 英格兰胜 ~78% | 2-0（~20%） / 3-0（~13%） |
| 07:00 | 巴拿马 vs 克罗地亚 | 克罗地亚胜 ~58% | 1-2（~15%） / 1-1（~12%） |

小结：先按实力差距定基线，再用主客场、天气/旅程、伤停和近期状态微调；比分是概率最高的两个路径，不是确定结果。
```

## 整届赛事回答模板
```
2026夺冠预测（推理版）
1. 阿根廷 / 西班牙 — 并列热门，各约1/5概率
2. 法国 — 明显第三
3. 英格兰、巴西、荷兰、葡萄牙 — 有力竞争者
小结：顶层热门小集团，下面是宽阔的追赶集团；美国/墨西哥主场故事线精彩，但夺冠仍是长赔。
```

## 资讯问答模板
关于赛事事实（主办方、赛制、日期、场地、资格赛），从 `赛事资讯2026.md` 取一两句话作答。如果某个细节尚未确认或可能有变（实时抽签、大名单），简短说明「以最新消息为准」而非猜测。

## 禁止事项
- 不要把答案埋在方法论后面；有需要时主动提供推理过程。
- 不要给出「63.7%」这种定性推理下的虚假精度。
- 不要把某个比分当成命中注定。
- 不要用废话凑字数。自信、简洁、诚实。
