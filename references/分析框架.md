# 分析框架（纯推理，不运行代码）

## 思维模型
1. 按 `实力评分指引.md` 提问时联网查询涉及球队的当前评分，映射到档位。
2. 根据下面的因素上下调整这个基线。
3. 将调整后的差距换算成结果倾向和两个候选比分。
4. 给出诚实的概率区间，点名2-3个驱动因素，收尾。

## 因素清单
逐项检查；大多数因素影响有限，少数几个能扭转结果。

- **基线实力**：双方各在哪个档位，差距大还是小？
- **近期状态**：最近6-10场表现、势头、信心、进失球数据。
- **阵容可用性**：伤病、停赛；一个核心球员（前锋/门将/组织者）缺阵可使球队下移近一个档位。
- **历史对阵与战术克制**：一方的强项是否正好打中另一方的弱点（高位压迫vs后场组织不稳、反击速度vs高防线、制空vs矮后卫）？
- **主场优势**：美国、加拿大、墨西哥是东道主。主场球迷加上熟悉环境约值三分之一到半个档位。墨西哥城海拔约2240米；美国夏季高温高湿削弱高位压迫球队体能。
- **赛事利害与动机**：必须赢的vs已经晋级可能轮换；垃圾时间比赛强度会下降。
- **疲劳与休息**：休息天数少、上轮打了加时、跨大陆城市间长途旅行——这些都有小额消耗。
- **定位球与门将**：强定位球体系或顶级门将能在胶着比赛中产生影响。
- **教练组与心理素质**：大赛经验和替补深度在后期关键。
- **天气与场地**：有屋顶/空调的美国球场、天然草皮vs临时人工草皮。

## 从因素到结论
将调整后的实力差距换算成结果倾向。单场比赛的粗略锚点（诚实区间，非精确数字）：

- 同档位，无额外优势：约38% / 26% / 36%（胜/平/负）；基本是抛硬币。
- 高出半档：热门约45-50%，平局约26%，冷门约25-28%。
- 高出一档：热门约55-62%，平局约23%，冷门约16-20%。
- 高出两档：热门约70-78%，平局约15-18%，冷门约8-12%。
- 高出三档及以上：热门约82-90%，但永远不要当作必然。

国际赛平局概率通常在22-28%之间；除非差距极大（此时平局概率下降），否则维持在此区间。

## 比分启发式
- 比分要与双方预期进球输出一致，而非单纯反映胜负赔率。大多数国际赛落在0-0、1-0、1-1、2-0、2-1。
- 每次比分预测给两个候选比分：首选 + 次选，并分别附粗略概率。单个准确比分命中率通常不高，常见首选比分约12-22%，次选约8-16%；强弱悬殊时热门小胜/完胜比分可略高，但避免超过约25%。
- 强热门对阵低位防守弱队：倾向1-0或2-0。
- 双方开放进攻：倾向2-1或2-2。
- 双方防守强或淘汰赛谨慎：倾向1-0或0-0（加时点球）。
- 两个比分要覆盖不同路径：例如热门胜 + 平局冷门路径，或小胜 + 扩大比分路径；避免列出十个比分。

## 淘汰赛vs小组赛
- 小组赛可以平局；平局是真实结果，各得一分。
- 淘汰赛不能平局：如果90分钟后评估接近，则延伸至加时和点球。点球接近掷硬币，更强或更沉稳的一方仅有小幅优势。「晋级倾向」要与90分钟结果分开给出。

## 理智检验
- 单场热门超过约85%需要真正悬殊的差距，否则调低。一场比赛方差极大。
- 48队赛事里最强队的夺冠概率通常12-25%，不会是多数。如果某队在预测里接近必胜，重新检视假设。
- 始终以概率表达结果。热门随时输球；永远不要给出100%。
